# CSS-Task-4

In this task, I have to create 2 level navigation bar where dropdown systems are applied. First, I have created a navigation bar, where the verical dropdown will be visible when hover the elements. And a horizontal dropdown will be visible when hovering the last element. In the second horizontal navbar, when hovering its elements, then the vertical dropdown will be visible.
